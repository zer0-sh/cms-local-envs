# Local CMS Environments (WordPress / Drupal)

Dockerized CMS environments for local development and testing, designed to replicate production-like configurations.

## 🚀 Overview

This project provides containerized setups for popular CMS platforms such as WordPress and Drupal, enabling consistent and portable development environments.

## 🐳 Features

- Docker Compose configurations for:
  - Web server (Nginx / Apache)
  - Database (MySQL / MariaDB / PostgreSQL)
  - CMS services (WordPress / Drupal)
- Production-like environment simulation
- Easy setup and teardown
- Isolated development environments

## 📦 Use Cases

- Local CMS development
- Testing plugins/themes/modules
- Reproducing production issues
- Rapid prototyping

## 🛠️ Tech Stack

- Docker
- Docker Compose
- WordPress / Drupal
- MySQL / MariaDB

## 📌 Notes

This repository is currently being adapted to support CMS-based environments. Some configurations may still reflect previous setups.

Dockerized CMS applications for local and testing environments, replicating production configurations.
